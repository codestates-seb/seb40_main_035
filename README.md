## 🗒 Commit / PR 컨벤션

- `feat` : 새로운 기능 추가
- `fix` : 버그 수정
- `design` : UI 변경
- `docs` : 문서 수정
- `refactor` : 코드 리팩토링
- `style` : 코드 포맷 수정
- `test` : 테스트 작성/수정
- `chore` : 기타 문제(의존성 문제, 설정 문제 등)
